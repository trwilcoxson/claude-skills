# ISO 27001:2022 Annex A Controls

## How to Use This Reference

ID format: `A.[theme].[number]` — 4 themes, 93 controls total. This is the 2022 revision which reorganized from 14 domains to 4 themes. When citing controls, use the exact ID format shown below.

New controls introduced in 2022 are marked with **(NEW)**.
